# HackResearch2025-LLMDillemma

For the UTRGV HackResearch 2025 Hackathon, we created a simulation for the volunteer's dilemma using LLM Agents.

The volunteer's dilemma is a game that models a situation in which each player can either make a small sacrifice that benefits everybody, or instead wait in hope of benefiting from someone else's sacrifice.
We decided to see if using several instances of Gemini AI could produce a nash equilibrium given the conditions.


Our limited findings found that they were not finding the true nash equilibrium but a different state of exploitation of a single agent for the benefit of the others.
